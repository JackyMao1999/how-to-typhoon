# 🌀 台风模拟与预警系统

基于 React、TypeScript 和 Leaflet 的交互式台风模拟与早期预警可视化系统。

## 功能特性

### 🌪️ 台风仿真引擎
- **物理机制**：5 种物理过程 — 大气环流突变、藤原效应、复杂地形阻挡、冷尾流效应、眼墙置换
- **MPI 强度模型**：最大潜势强度公式计算真实强度上限
- **快速增强**：环境条件适宜时可触发 3-7 倍加速增强
- **动态引导流**：多层引导气流 + 随机扰动
- **智能自动生成**：环境条件驱动的台风生成，支持连续自动仿真

### 🗺️ 离线矢量地图
- **完全离线**：基于 Leaflet + GSHHG 海岸线数据，无需 API Key 或在线瓦片服务
- **GSHHG 海岸线**：全球自洽高分辨率海岸线数据库
- **经纬网格**：经纬线网格 + 赤道和回归线标注
- **在线增强**：可选 CartoDB 暗色瓦片（免费、无 API Key）

### 📊 实时监控
- **台风监控面板**：当前强度、风圈半径、气压、位置、移动状态
- **影响因素显示**：实时解释当前影响台风强度的物理因子
- **登陆点追踪**：自动检测首次登陆，地图标记 + 面板显示

### 🌤️ 真实天气模式
- **实时数据获取**：通过 Open-Meteo API 获取当前位置真实海温、风切变、湿度、气压
- **自动刷新**：每 10 分钟或台风移动 >1° 时自动更新
- **无需 API Key**：完全免费的开放 API

### ⚠️ 区域预警系统
- **自定义监测区域**：预置华南沿海、台湾、菲律宾、日本等关注区域
- **四色预警**：蓝色(注意)、黄色(预警)、橙色(警戒)、红色(紧急)
- **Toast 通知**：台风接近监测区域时自动弹出提示（含音效 + 闪烁）

### 📜 历史数据
- **内置样例**：海燕(2013)、山竹(2018)、利奇马(2019)、摩羯(2024)
- **回放时间轴**：完整时间轴回放，支持自动播放
- **数据导入**：支持 CSV 和 JSON 格式导入自定义台风数据

### 🎮 控制调节
- **环境参数**：海温、风切变、湿度、海洋热含量、摩擦系数、副热带高压
- **图层切换**：风圈、路径预测、集合预报、海岸线、登陆点标记
- **极端模式**：19 级风暴模拟，用于极端情景演示
- **自动模式**：环境参数随纬度和季节自动调整

## 技术栈

- **框架**：React 18 + TypeScript
- **状态管理**：Zustand
- **地图引擎**：Leaflet
- **WebGL 渲染**：自定义台风眼发光效果
- **海岸线数据**：GSHHG（NOAA 全球自洽高分辨率海岸线数据库）
- **天气 API**：Open-Meteo（免费，无需 API Key）

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 使用指南

1. 打开右上角「控制台」
2. 调整环境参数或开启「🌤 真实天气模式」获取实时数据
3. 仿真默认自动开始，生成随机台风
4. 点击地图任意位置可在该处生成台风
5. 开启「自动生成」实现连续仿真
6. 在监控面板查看实时状态和影响因素
7. 设置监测区域获取台风接近预警

## 控制台参考

| 分区 | 功能 | 说明 |
|------|------|------|
| **Environment** | 环境参数 | 海温、风切变等自动/手动模式 |
| **Intensity Factors** | 高级强度因子 | 风切变、热含量、湿度、台风尺度 |
| **Extreme Mode** | 极端模式 | 19 级风暴模拟 |
| **Physics** | 物理机制开关 | 环流突变、藤原效应、地形阻挡、冷尾流、眼墙置换 |
| **Layers** | 图层切换 | 风圈、路径、集合预报、海岸线 |
| **Alert Regions** | 监测区域 | 选择需要关注的海域 |
| **Season** | 季节模式 | 春/夏/秋/冬 环境预设 |

## 数据来源

- **海岸线**：GSHHG（NOAA）
- **天气（可选）**：Open-Meteo API
- **地图瓦片（可选）**：CartoDB Dark Matter（免费）
